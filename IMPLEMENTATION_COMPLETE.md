## 🏗️ Technical Implementation

### Architecture
```
User Interface (React)
    ↓
12 Union Cards + 4-Step Form + Anonymous Flag
    ↓
Union ID Mapping (string → ObjectId)
    ↓
Backend API (Express.js)
    ↓
MongoDB Database
```

### Union Mapping System
- **Frontend**: Uses simple string IDs ('1'-'12') for hardcoding
- **Mapping Layer**: Converts to MongoDB ObjectIds automatically
- **Backend**: Stores with ObjectId references
- **Result**: Seamless, error-proof integration

### Problem Submission Flow
```

   ↓
2. Step 1: Select Union (12 colorful cards)
   ↓

   ↓
```
