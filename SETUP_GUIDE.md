# ত্রিশাল নাগরিক ম্যাপ - সেটআপ গাইড

## 📋 প্রাথমিক প্রস্তুতি

### প্রয়োজনীয় সফটওয়্যার ইনস্টল করুন

1. **Node.js ডাউনলোড করুন**
   - https://nodejs.org/ থেকে LTS সংস্করণ ডাউনলোড করুন
   - ইনস্টলার চালান এবং সমস্ত ডিফল্ট বিকল্প গ্রহণ করুন

2. **Git ইনস্টল করুন** (ঐচ্ছিক কিন্তু সুপারিশকৃত)
   - https://git-scm.com/ থেকে ডাউনলোড করুন

3. **কোড এডিটর**
   - VS Code সুপারিশকৃত: https://code.visualstudio.com/

### Node.js ইনস্টলেশন যাচাই করুন

কমান্ড প্রম্পট খুলুন এবং চালান:

```bash
node --version
npm --version
```

সংস্করণ সংখ্যা প্রদর্শিত হওয়া উচিত।

## 🗄️ MongoDB সেটআপ

1. **MongoDB Atlas অ্যাকাউন্ট তৈরি করুন**
   - https://www.mongodb.com/cloud/atlas এ যান
   - বিনামূল্যে অ্যাকাউন্ট তৈরি করুন
   - ইমেল যাচাই করুন

2. **প্রথম ক্লাস্টার তৈরি করুন**
   - "Create a Deployment" ক্লিক করুন
   - "Free" প্ল্যান নির্বাচন করুন
   - "Create Cluster" ক্লিক করুন

3. **ডাটাবেস অ্যাক্সেস সেটআপ করুন**
   - বাম মেনু থেকে "Database Access" ক্লিক করুন
   - "Add New Database User" ক্লিক করুন
   - ব্যবহারকারীনাম: `trishal_user`
   - পাসওয়ার্ড: `trishal123` (অথবা কাস্টম নির্বাচন করুন)
   - "Add User" ক্লিক করুন

4. **সংযোগ স্ট্রিং পান**
   - "Connect" বাটন ক্লিক করুন
   - "Connect your application" নির্বাচন করুন
   - সংযোগ স্ট্রিং কপি করুন

## 🚀 প্রকল্প সেটআপ

### ধাপ ১: ফোল্ডার খুলুন

1. প্রজেক্ট ফোল্ডার (`d:\trishal`) খুলুন
2. এই ফোল্ডার থেকে কমান্ড প্রম্পট খুলুন

### ধাপ ২: Backend সেটআপ

```bash
# Backend ফোল্ডারে যান
cd backend

# সমস্ত প্যাকেজ ইনস্টল করুন
npm install

# অপেক্ষা করুন - এটি কয়েক মিনিট সময় নিতে পারে
```

**.env ফাইল তৈরি করুন:**

1. `backend` ফোল্ডারে `.env` নামের ফাইল তৈরি করুন
2. নিম্নলিখিত কন্টেন্ট যোগ করুন:

```env
MONGODB_URI=mongodb+srv://trishal_user:trishal123@cluster0.mongodb.net/trishal-civic?retryWrites=true&w=majority
PORT=5000
NODE_ENV=development
JWT_SECRET=your_jwt_secret_key_12345
ADMIN_EMAIL=admin@trishal.local
ADMIN_PASSWORD=admin123
```

**Backend চালু করুন:**

```bash
npm start
```

আপনি দেখবেন:
```
Server running on port 5000
MongoDB Connected
```

### ধাপ ৩: Frontend সেটআপ

**নতুন কমান্ড প্রম্পট উইন্ডো খুলুন** এবং:

```bash
# প্রকল্প ফোল্ডারে ফিরে আসুন
cd d:\trishal

# Frontend ফোল্ডারে যান
cd frontend

# সমস্ত প্যাকেজ ইনস্টল করুন
npm install

# অপেক্ষা করুন - এটি আরও বেশি সময় নিতে পারে
```

**.env ফাইল তৈরি করুন:**

1. `frontend` ফোল্ডারে `.env` নামের ফাইল তৈরি করুন
2. নিম্নলিখিত যোগ করুন:

```env
REACT_APP_API_URL=http://localhost:5000/api
```

**Frontend চালু করুন:**

```bash
npm start
```

এটি স্বয়ংক্রিয়ভাবে `http://localhost:3000` এ আপনার ব্রাউজার খুলবে।

## ✅ যাচাই করুন

1. **ব্রাউজার খুলুন**: `http://localhost:3000`
2. আপনি হোম পেজ দেখতে পাবেন যেখানে ত্রিশাল উপজেলার মানচিত্র রয়েছে
3. সকল লিঙ্ক কাজ করে কিনা তা পরীক্ষা করুন

## 🔐 অ্যাডমিন প্যানেল অ্যাক্সেস

1. ব্রাউজারে যান: `http://localhost:3000/admin/login`
2. ডেমো শংসাপত্র ব্যবহার করুন:
   - ইমেল: `admin@trishal.local`
   - পাসওয়ার্ড: `admin123`
3. ড্যাশবোর্ড অ্যাক্সেস করুন

## 🐛 সাধারণ সমস্যা সমাধান

### সমস্যা: "Port 5000 already in use"
**সমাধান:**
```bash
# Windows এ বিদ্যমান প্রক্রিয়া খুঁজুন
netstat -ano | findstr :5000

# প্রক্রিয়া বন্ধ করুন (PID দিয়ে)
taskkill /PID <PID> /F

# অথবা ভিন্ন পোর্ট ব্যবহার করুন
PORT=5001 npm start
```

### সমস্যা: "MongoDB connection failed"
**সমাধান:**
1. MongoDB Atlas এ লগইন করুন
2. ক্লাস্টার চলছে কিনা চেক করুন
3. IP হোয়াইটলিস্ট যাচাই করুন (0.0.0.0/0 যোগ করুন ডেভেলপমেন্টের জন্য)
4. সংযোগ স্ট্রিং পুনরায় কপি করুন

### সমস্যা: "React app not loading"
**সমাধান:**
1. npm install সম্পূর্ণ হয়েছে কিনা চেক করুন
2. পোর্ট 3000 ফ্রি আছে কিনা চেক করুন
3. `npm start` পুনরায় চালান

### সমস্যা: "Map not showing"
**সমাধান:**
1. ইন্টারনেট সংযোগ যাচাই করুন
2. ব্রাউজার কনসোল খুলুন (F12) এবং ত্রুটি দেখুন
3. Leaflet প্যাকেজ ইনস্টল করা আছে কিনা চেক করুন
