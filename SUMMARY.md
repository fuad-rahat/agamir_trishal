## 🚀 এখনই শুরু করুন (3 সহজ ধাপ)

### ধাপ 1: Backend চালু করুন
```bash
cd backend
npm install
npm start
```

### ধাপ 2: Frontend চালু করুন (নতুন কমান্ড প্রম্পট)
```bash
cd frontend
npm install
npm start
```

### ধাপ 3: ব্রাউজার খুলুন
```
http://localhost:3000
```
