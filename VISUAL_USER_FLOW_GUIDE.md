## 🔄 DATA FLOW DIAGRAM

```
User Interaction:
                    ┌─────────────────┐
                    │ Click Union     │
                    │ on Map          │
                    └────────┬────────┘
                             │
                             ▼
          ┌──────────────────────────────────┐
          │  ReactRouter Navigate            │
          │  → /union/:unionId               │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  UnionDetailPage Component       │
          │  Fetches data from API           │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  GET /api/unions/:id             │
          │  (No auth required)              │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Database Query                  │
          │  Return Union Data               │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Render Page with 5 Tabs         │
          │  Display all information         │
          └──────────────────────────────────┘


Admin Interaction:
                    ┌─────────────────┐
                    │ Admin Login     │
                    │ /admin/login    │
                    └────────┬────────┘
                             │
                             ▼
          ┌──────────────────────────────────┐
          │  AdminLoginPage Component        │
          │  Submit Email & Password         │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  POST /api/auth/login            │
          │  Verify credentials              │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Generate JWT Token              │
          │  (7-day expiration)              │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Save Token to localStorage      │
          │  Redirect to Dashboard           │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  AdminDashboardPage              │
          │  Load unions from API            │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Display Union Grid              │
          │  Show status indicators          │
          └────────────┬─────────────────────┘
                       │
                       ▼ Select Union
          ┌──────────────────────────────────┐
          │  Show Edit Forms with Tabs       │
          │  Admin adds information          │
          └────────────┬─────────────────────┘
                       │
                       ▼ Submit Form
          ┌──────────────────────────────────┐
          │  POST /api/unions/:id/{category} │
          │  Include JWT Token in Header     │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Verify Token & Permissions      │
          │  Add data to Database            │
          └────────────┬─────────────────────┘
                       │
                       ▼
          ┌──────────────────────────────────┐
          │  Success Response                │
          │  Update Preview Section          │
          └──────────────────────────────────┘
```
